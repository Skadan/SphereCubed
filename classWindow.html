Detailed Description

Window class.

This class declaration encapsulates the Window functionality. One Window is instantiated to display the Graphic User Interface (GUI).

Window::Window ( QWidget *  parent = 0)

The Graphical User Interface (GUI) Window execution begins here. Calling this constructor is required to setup the Window. Passes parent argument onto base class constructor.
